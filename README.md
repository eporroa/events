# Event Management Platform

## Project Structure
- Monorepo using Turborepo
- Microservices architecture
- Docker-based development environment

## Technologies
- Backend: Nest.js
- Frontend: Next.js
- Database: MongoDB
- State Management: Zustand
- API: GraphQL
- Monorepo: Turborepo

## Prerequisites
- Docker
- Docker Compose
- Node.js 20+
